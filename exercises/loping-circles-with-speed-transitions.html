Loping Circles with Speed Transitions

Focus: Speed Control, Collection, Adjustability

Equipment: 4 cones or markers

Step-by-Step Instructions

  1. Lope a large circle at a forward, working pace.
  2. After 2 revolutions, collect to a slow lope for 2 more revolutions on the same circle.
  3. The difference should be obvious: forward lope covers more ground; slow lope is shorter and more uphill.
  4. Open back to the forward lope for 2 revolutions.
  5. The circle size should stay the same in both speeds — only the stride length changes.
  6. Practice on both leads for equal development.

Common Mistakes to Avoid

  • Changing the circle size when changing the speed (bigger for fast, smaller for slow).
  • The slow lope breaking into a jog because there's not enough impulsion.
  • The fast lope becoming a flat run instead of a quality forward lope.
  • Not enough contrast between the two speeds.

Variations & Progressions

  • Add a third speed: very slow (collected), working, and extended for three-speed practice.
  • Change speed every revolution instead of every 2.
  • Combine speed changes with lead changes on a figure-eight.
Tip: Speed control is the most important skill in western performance. A horse with only one speed wins nothing.
