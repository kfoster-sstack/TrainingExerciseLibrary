Tail Pull for Back Engagement

Focus: Core Strength, Back Health, Recovery

Equipment: None required

Arena Diagram

Step-by-Step Instructions

  1. Stand behind and slightly to one side of your horse (never directly behind).
  2. Grasp the tail firmly (not the hair alone — grab the dock) about 6 inches from the base.
  3. Gently pull the tail toward you. The horse should resist, engaging its back and core muscles.
  4. Hold the pull for 5-10 seconds, then release. The horse should step slightly sideways.
  5. Repeat 3-5 times from each side.
  6. This exercise isometrically strengthens the muscles along the spine.

Common Mistakes to Avoid

  • Standing directly behind the horse (dangerous position).
  • Pulling too hard or jerking the tail.
  • Grabbing only the hair, which can pull out.
  • Not alternating sides for equal engagement.

Variations & Progressions

  • Combine with belly lifts for a comprehensive core workout.
  • Use as part of a pre-ride warm-up routine.
  • Add to a rehabilitation program for horses with sacroiliac issues.
Tip: Tail pulls engage the entire topline in an isometric contraction. It's like a plank exercise for horses.
